# 用户中心与权限

## 这篇文档解决什么问题

用户中心用于管理本地登录账号、角色权限、功能入口可见性和操作偏好。平台没有开放注册入口，账号由超级管理员统一创建和分配。

## 适用对象

- 超级管理员：创建用户、设置口令、分配角色和权限。
- 发布员：使用与发布相关的功能入口。
- 素材维护员：使用素材和视频生成相关入口。
- 数据监控员：使用数据统计和通知相关入口。

## 登录规则

- 超级管理员使用固定管理员口令登录。
- 其他角色账号必须由超级管理员创建。
- 其他角色的初始口令和重置口令只能由超级管理员分配。
- 退出后会进入独立登录页，控制台侧栏和主界面会隐藏。

## 角色说明

### 超级管理员

拥有全部功能入口权限，可以创建账号、添加角色、设置口令、调整角色权限。

### 发布员

默认可访问总览、账号矩阵、公共设置、任务中心、视频生成、通知中心和帮助文档。

### 素材维护员

默认可访问总览、账号矩阵、视频生成、通知中心和帮助文档。

### 数据监控员

默认可访问总览、数据统计、通知中心和帮助文档。

## 账号分配流程

1. 超级管理员进入用户中心。
2. 在账号分配区填写姓名。
3. 选择角色。
4. 填写初始口令。
5. 点击添加账号。
6. 用户通过独立登录页选择账号并输入口令登录。

## 口令重置流程

1. 超级管理员进入用户中心。
2. 找到目标用户行。
3. 在“设置/重置口令”输入新口令。
4. 点击保存口令。
5. 通知用户使用新口令登录。

## 常见问题

### 为什么普通用户看不到某些菜单？

菜单显示由角色权限决定。需要超级管理员在功能入口权限中给该角色勾选对应入口。

### 为什么没有注册按钮？

平台采用管理员分配账号模式，避免运营账号和权限失控。

### 为什么超级管理员口令不能在页面修改？

超级管理员口令属于系统级配置，当前不在普通 UI 中修改，避免误操作导致无法管理平台。

## 操作检查清单

- 新账号已分配角色。
- 新账号已设置初始口令。
- 角色权限符合岗位职责。
- 普通用户退出后能进入独立登录页。
- 普通用户只能看到授权入口。
