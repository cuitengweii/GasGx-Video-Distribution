# 视频生成工作台

## 用途

视频生成工作台用于从本地分类素材生成矩阵短视频，包含素材读取、封面、文案、背景音乐、组合结构和批量导出。

## 主要功能

- 按分类目录读取视频碎片素材。
- 设置视频时长、第一屏封面、字幕文案和输出目录。
- 使用本地背景音乐库或 Pixabay 行业音乐。
- 调整素材组合结构和分类轮换算法。
- 生成前确认算法框架、启用分类和导出结构。

## 推荐流程

1. 准备分类素材目录。
2. 检查背景音乐库是否有可用 MP3。
3. 设置生成结构和输出路径。
4. 预览关键参数后开始生成。
5. 生成后检查输出目录和封面文案。

## 注意事项

- 本地 BGM 目录为空时，系统只能按当前可用来源生成。
- 生成任务失败时，先检查素材路径、输出目录权限和 FFmpeg 可用性。
