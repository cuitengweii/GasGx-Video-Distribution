# 发布任务与队列

## 用途

任务中心用于创建和查看发布、评论、私信、统计等自动化任务，并跟踪 pending、running、success、failed 等状态。

## 主要功能

- 按账号、平台和任务类型创建任务。
- 查看任务队列、运行状态和失败记录。
- 避免同一账号、同一平台、同一任务类型重复加入 pending 或 running 队列。
- 配合公共设置读取发布素材目录和平台策略。

## 处理失败任务

- 登录失效：回到账号矩阵恢复 profile 登录。
- 素材不足：补充素材目录或调整公共设置。
- 上传超时：提高平台上传超时，或检查网络和文件大小。
- 执行冲突：等待当前任务结束，避免重复占用同一 profile。
