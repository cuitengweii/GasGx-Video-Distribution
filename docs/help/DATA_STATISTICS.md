# 数据统计看板

## 用途

数据统计看板用于从客户视角查看短视频矩阵运营效果，包括账号、内容、流量来源、营销转化、执行数据和风险趋势。

## 主要模块

- 总览指标：播放、互动、线索、转化等核心指标。
- 账号矩阵统计：账号分层、完播率、互动率和异常占比。
- 作品内容数据：内容排行、发布时段和 TOP 内容。
- 流量来源分析：不同来源的占比与效果。
- 营销转化：从播放到有效线索的漏斗。
- 风险预警：异常账号、素材不足、趋势下滑等提示。

## 注意事项

- 当前页面结构已就绪，部分数据仍可能来自样例或后端聚合接口。
- 数据库模式优先使用 `dashboard_summary()` RPC；老客户库未执行 SQL 时会回退到 REST 聚合。
