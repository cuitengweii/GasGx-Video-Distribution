# 数据统计看板

## 这篇文档解决什么问题

数据统计看板用于复盘账号矩阵的播放、互动、流量来源和商业转化。它帮助运营人员判断哪些账号、平台、内容主题和发布时间段更有效。

## 适用对象

- 数据监控员：查看趋势、异常和转化指标。
- 发布员：根据内容表现调整发布节奏。
- 超级管理员：评估账号矩阵价值和资源投入。

## 核心指标

- 播放量：内容触达规模。
- 点赞、评论、分享：内容互动质量。
- 私信、线索：潜在客户转化。
- 完播率：内容是否能被完整观看。
- 异常账号占比：账号矩阵健康状态。

## 页面模块

### 账号矩阵统计

按账号和平台查看表现，适合判断哪些账号需要继续投入、养号或暂停。

### 作品内容数据

查看内容 TOP 列表，识别高表现主题、标题结构、封面形式和发布时间段。

### 流量来源分析

分析流量来自推荐、搜索、主页、转发还是其他入口，帮助判断内容分发方式。

### 商业转化

关注主页访问、私信咨询、有效线索和意向客户。对 B2B 场景来说，转化指标比单纯播放量更重要。

### 风险预警

查看账号异常、播放下滑、互动异常或任务失败趋势。

## 数据复盘流程

1. 先看时间范围，确认统计周期。
2. 按平台筛选，比较不同平台差异。
3. 按账号筛选，找出强账号和弱账号。
4. 查看内容 TOP，提取可复用结构。
5. 查看转化漏斗，判断内容是否带来有效询盘。
6. 把结论反馈给素材和发布策略。

## 常见问题

### 播放高但询盘少怎么办？

检查内容是否过于泛流量，是否缺少明确产品场景、案例细节和行动引导。

### 互动低是否一定代表内容不好？

不一定。B2B 内容可能互动低但线索质量高。需要结合私信、主页访问和有效线索判断。

### 数据为空怎么办？

确认统计任务是否执行、账号是否有发布内容、数据库是否初始化，并检查筛选条件是否过窄。

## 操作检查清单

- 明确统计周期。
- 分平台对比。
- 分账号复盘。
- 记录高表现内容模板。
- 把复盘结论反馈到素材和发布计划。
