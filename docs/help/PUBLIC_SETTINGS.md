# 公共设置与平台配置

## 用途

公共设置用于维护全局素材目录、默认发布方式、话题、上传超时，以及每个平台的独立发布配置。

## 配置层级

1. 平台独立配置优先。
2. 未设置平台独立配置时继承全局公共配置。
3. 全局未设置时使用代码默认值。

## 主要功能

- 设置发布素材目录和常用路径候选。
- 设置默认发布方式、全局话题和上传超时。
- 为视频号等平台配置短标题、位置、合集、原创声明、可见范围和评论权限。
- 保存后供任务中心和发布执行层读取。

## 注意事项

- 修改素材目录后，需要确认目录里有可用视频或素材文件。
- 平台独立文案会覆盖全局文案，不要在两处写冲突内容。
